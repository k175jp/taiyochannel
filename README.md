# taiyochannel
taiyochannelはやられ掲示板アプリです。

# .envファイル
docker-compose.ymlと同じディレクトリに配置
```
MYSQL_DATABASE=taiyochannel_database
MYSQL_USER=taiyochannel_user
MYSQL_PASSWORD=taiyochannel_password
MYSQL_ROOT_PASSWORD=taiyochannel_password
TZ=Asia/Tokyo
LANG=ja_JP.UTF-8
LANGUAGE=ja_JP:ja
DB_HOST=db
DB_PORT=3306
```
